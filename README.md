# Collection of tools for VM management and communication

## Tunnel
This tool allows establishing communication between linux machines inside networks behind NATs and not published to the Internet and a service outside.

## VM
This is a tool for managing virtual machines using QEMU. This tools allows the easy creation of VMs using cloud images using specs written in JSON.

## VPN
This are the instructions and templates to establish an IPSec VPN using certificate authentication and only one of them has a static IP address and DNS record registered.
